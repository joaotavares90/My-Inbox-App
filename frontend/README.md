# My Inbox App - Frontend

A simple React application for creating, viewing, and managing messages.

## Requirements Fulfilled

âœ… **3 Pages:**

- Landing page
- Detail page
- Create page

**Features:**

- Messages display date and subject in the list
- Click on message navigates to detail page
- Delete messages from detail page
- Create messages with subject and text
- Automatic redirect to landing page after creation

## Tech Stack

- React 18
- React Router v6
- Vite (build tool)
- Plain CSS (no frameworks)

## Setup & Run

# Install dependencies

npm install

# Run development server (port 3000)

npm run dev
